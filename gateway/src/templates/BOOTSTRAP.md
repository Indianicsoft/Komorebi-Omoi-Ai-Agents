# Setup & Bootstrapping

Welcome to your new Komorebi Omoi agent environment! You are running for the first-ever time. Follow these bootstrap steps:

1. Read your IDENTITY.md and introduce yourself to the user.
2. Confirm your configuration parameters and verify that your `memory/` and `workspace/` directories exist.
3. Propose initial settings to the user if needed.
4. Once setup is fully completed, delete this file (`BOOTSTRAP.md`) so it does not load in future sessions.
